ТЗ для приложения заметок на Cordova с поддержкой WebDAV

1. Цель: Создать приложение заметок на Cordova с синхронизацией через WebDAV и минималистичным дизайном на Bootstrap.

2. Функционал:
   - Создание, редактирование, удаление, просмотр заметок с визуальным редактором (WYSIWYG).
   - Поиск заметок по ключевым словам.
   - Синхронизация с WebDAV (CRUD-операции: создание, чтение, обновление, удаление).
   - Оффлайн-режим с автоматической синхронизацией.

3. Технические детали:
   - Платформа: Cordova (iOS, Android).
   - Языки и фреймворки: HTML, CSS, JS (Bootstrap 5, опционально jQuery или другие легковесные фреймворки).
   - WebDAV: HTTP-запросы через cordova-plugin-file-transfer.
   - Локальное хранение: IndexedDB.
   - Структура хранения:
     - Локально: заметки сохраняются в IndexedDB с ключами "notes/[id]", изображения в "images/[id]".
     - Удаленно (WebDAV): заметки в папке "/notes/[id].json", изображения в "/images/[id].jpg".

4. Дизайн:
   - Минималистичный интерфейс на Bootstrap 5:
     - Кнопка "Создать заметку".
     - Список заметок (сетка/список) с кнопками "Редактировать", "Удалить", "Просмотреть".
     - Визуальный редактор (форматирование: жирный, курсив, списки).
     - Строка поиска с фильтрацией.
     - Кнопка переключения тем (светлая/темная).
   - Адаптивность: корректное отображение на экранах от 320px до 1200px.

5. Настройки:
   - URL WebDAV-сервера.
   - Логин и пароль для WebDAV (опционально).
   - Частота автосинхронизации (выкл./5 мин/15 мин).
   - Выбор темы (светлая/темная/системная).
   - Язык интерфейса (русский/английский).
   - Экспорт/импорт заметок в JSON.

6. Дополнительные идеи:
   - Поддержка прикрепления изображений к заметкам (сжатие перед загрузкой на WebDAV).
   - Уведомления о статусе синхронизации.
   - Горячие клавиши для редактора (Ctrl+B, Ctrl+I и т.д.).
   - Архив заметок с возможностью восстановления (локально: "archive/[id]", удаленно: "/archive/[id].json").

7. Результат:
   - Приложение заметок на Cordova с минималистичным дизайном на Bootstrap, кнопками для создания, редактирования, удаления, просмотра заметок, поиска, переключения тем и синхронизацией через WebDAV.
